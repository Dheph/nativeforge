# NativeForge

A modular, highly scalable CLI for scaffolding React Native and Expo applications with domain-driven design, robust infrastructure setups (Firebase, Supabase, etc), and ready-to-use microcomponents.

## Monorepo Structure

- `packages/cli`: The NativeForge CLI engine.
- `packages/registry`: The truth source for structural templates, modules and UI components.
- `apps/docs`: The official documentation website.
- `apps/sandbox`: Test environment for React Native / Expo.
